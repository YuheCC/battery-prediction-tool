# 早期生命预测工具使用指南

## 功能概述

早期生命预测工具是一个基于机器学习的电池寿命预测系统，能够通过分析电池的早期循环数据来预测其完整寿命。

## 主要特性

- 📊 **多特征分析**: 基于容量、电压、温度、循环次数等多个特征进行预测
- 🎯 **智能算法**: 使用机器学习算法进行准确的寿命预测
- 📈 **置信度评估**: 为每个预测结果提供置信度评分
- 📋 **详细报告**: 生成包含预测结果、特征分析和建议的完整报告
- 💾 **历史记录**: 自动保存预测历史，支持查看和对比

## 使用流程

### 1. 数据上传
- 支持 CSV 和 Excel 格式文件
- 文件大小限制：10MB
- 必需字段：Barcode（电池条码）
- 可选字段：Cycle（循环次数）、Capacity（容量）、Voltage（电压）、Temperature（温度）

### 2. 数据格式要求

#### CSV 文件格式示例：
```csv
Barcode,Cycle,Capacity,Voltage,Temperature
BAT001,5,0.985,3.65,25.2
BAT002,8,0.972,3.62,26.1
BAT003,12,0.958,3.58,24.8
```

#### 字段说明：
- **Barcode**: 电池唯一标识符
- **Cycle**: 当前循环次数
- **Capacity**: 容量保持率（0-1之间）
- **Voltage**: 电池电压（V）
- **Temperature**: 工作温度（°C）

### 3. 预测分析
- 点击"开始预测分析"按钮
- 系统将自动分析数据并生成预测结果
- 处理时间约2-3秒

### 4. 结果解读

#### 预测结果包含：
- **循环寿命**: 预测的电池完整循环次数
- **置信度**: 预测结果的可靠性评分（0-100%）
- **特征数据**: 用于预测的原始特征值
- **解释说明**: 预测结果的详细解释

#### 置信度颜色说明：
- 🟢 **绿色 (80%+)**: 高置信度，预测结果可靠
- 🟡 **黄色 (60-79%)**: 中等置信度，建议关注
- 🔴 **红色 (<60%)**: 低置信度，需要更多数据

## 预测算法说明

### 特征权重：
1. **容量保持率** (权重: 40%): 最重要的预测指标
2. **电压稳定性** (权重: 25%): 反映电池健康状态
3. **温度影响** (权重: 20%): 环境因素对寿命的影响
4. **循环次数** (权重: 15%): 当前使用程度

### 预测逻辑：
- 容量保持率 > 95%: 寿命延长 150 次
- 容量保持率 90-95%: 寿命延长 100 次
- 容量保持率 85-90%: 寿命延长 50 次
- 容量保持率 < 85%: 寿命减少 50 次

## 使用建议

### 数据质量要求：
1. **数据完整性**: 确保关键字段不缺失
2. **数据准确性**: 使用可靠的测试设备获取数据
3. **样本数量**: 建议至少10个样本以获得更准确的预测

### 最佳实践：
1. **定期预测**: 建议每季度进行一次预测分析
2. **数据更新**: 及时更新电池使用数据
3. **结果验证**: 对比实际使用情况验证预测准确性

## 故障排除

### 常见问题：

**Q: 文件上传失败**
A: 检查文件格式是否为CSV或Excel，文件大小是否超过10MB

**Q: 预测结果不准确**
A: 确保数据质量，增加样本数量，检查特征数据的准确性

**Q: 置信度较低**
A: 可能是数据不足或特征缺失，建议补充更多相关数据
