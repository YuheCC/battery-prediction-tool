# Ask优化-待评审

## 项目简介

这是一个基于Web的分子宇宙应用优化项目，专注于提升用户界面和交互体验。

## 项目信息

- **项目名称**: Ask优化-待评审
- **版本**: 1.0.0
- **状态**: 待评审
- **开发环境**: Node.js + Express

## 功能特性

- 🗺️ **分子地图可视化** - 基于ECharts的分子分布图
- 💬 **智能聊天界面** - 优化的Ask功能
- 🔍 **搜索与过滤** - 分子搜索和筛选功能
- 👤 **用户管理** - 登录、注册、密码重置
- 📱 **响应式设计** - 适配多种设备

## 技术栈

- **前端**: HTML5, CSS3, JavaScript, ECharts
- **后端**: Node.js, Express
- **数据**: JSON文件存储
- **样式**: 自定义CSS框架

## 快速开始

1. **安装依赖**
   ```bash
   npm install
   ```

2. **启动服务**
   ```bash
   node server.js
   ```

3. **访问应用**
   打开浏览器访问: http://localhost:3006
